# kdev-resume
This project was built by Vue 3.2.1.

## Local environment
```
$ git clone https://github.com/kdevsoftware/kdev-resume.git
$ cd kdev-resume
$ npm install
$ npm run serve
```
Navigate to `http://localhost:8080/`.

## Deploying to github page
```
$ npm run deploy
```
Navigate to `https://kdevsoftware.github.io/kdev-resume/`.
